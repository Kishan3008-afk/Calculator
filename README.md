## ğŸ”§ GitHub Pages Setup

### Method 1: Automatic (Recommended)
1. Fork the repository on GitHub
2. Go to **Settings â†’ Pages**
3. Select **Main** branch as source
4. GitHub automatically builds and deploys
5. Your site is live at `https://YOUR-USERNAME.github.io/Calculator/`

### Method 2: Manual Deployment

**Prerequisites:**
- Git installed on your system
- A GitHub account
- GitHub CLI (optional but recommended)

**Steps:**

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Kishan3008-afk/Calculator.git
   cd Calculator
   ```

2. **Make changes (optional):**
   - Edit HTML files as needed
   - Test locally by opening `index.html` in your browser

3. **Push to your fork:**
   ```bash
   git add .
   git commit -m "Update calculator features"
   git push origin main
   ```

4. **Enable GitHub Pages:**
   - Go to repository Settings
   - Navigate to "Pages"
   - Set source to `main` branch
   - Save

5. **Access your site:**
   - Wait 1-2 minutes for deployment
   - Visit `https://YOUR-USERNAME.github.io/Calculator/`
